# Library Database Management System (Oracle PL/SQL)

This project is an academic database system designed to manage the activity of a library.

## Technologies
- Oracle SQL
- PL/SQL
- SQL Developer

## Description
A relational database designed to manage:
- books
- authors
- publishers
- readers
- employees
- loans

## Implemented Features
- Table relationships using primary and foreign keys
- Stored procedures and user-defined functions
- PL/SQL packages:
  - Pachet_Biblioteca
  - Pachet_Angajati
- Triggers for:
  - automatic commission setting
  - salary validation
  - loan date verification
- Cursor usage (explicit and implicit)
- Exception handling using custom errors
- Archiving unused books
- Employee performance reporting
- Active reader tracking
